# **App Name**: PrdEngine

## Core Features:

- Prompt Architect: Uses AI as a tool to transform a simple app concept into a structured five-section PRD, automatically generating sections from Overview to Phases.
- Dynamic Form Intake: A focused user interface to capture essential project data required for the document synthesis.
- Split-Screen Live Editor: View and edit the generated Markdown side-by-side with its rendered output in real-time.
- Phase Planner: Interactive UI for visualizing and adjusting project development phases and milestones within the document structure.
- Direct Markdown Export: Seamlessly download the finalized document as a standard .md file compatible with any editor.

## Style Guidelines:

- Primary color: Deep Amethyst (#5E5AE1) symbolizing modern intelligence and authority.
- Background color: Dark Space (#17161C), providing a professional high-contrast dark-mode workspace.
- Accent: Cool Cobalt (#2E60B8) used for primary call-to-actions and structural markers.
- A pairing of 'Space Grotesk' for modern, tech-forward headlines and 'Inter' for highly legible body content.
- Sharp, linear vector icons that reflect technical precision and architectural planning.
- Modular grid-based dashboard with significant whitespace between PRD sections to prevent cognitive overload.
- Smooth, fluid transitions for generative AI text appearing, emulating a professional writer's workflow.
